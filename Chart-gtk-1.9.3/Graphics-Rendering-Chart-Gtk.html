renderableToWindow :: Renderable a -> Int -> Int -> IO () Source #

Display a renderable in a gtk window.

Note that this is a convenience function that initialises GTK on its first call, but not subsequent calls. Hence it's unlikely to be compatible with other code using gtk. In that case use createRenderableWindow.
